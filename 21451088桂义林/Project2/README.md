# iPhone开发第二次作业
======
- 作业内容：实现一个计算器
- 平台：iOS
- 编程语言：Objective-C
- 测试环境：iOS Simulator - iPhone 6 / iOS 8.0
- 实现功能说明：
  - '+'：加号
  - '-'：减号
  - 'x'：乘号
  - '/'：除号
  - '%'：取余数
  - '(', ')'：左、右括号
  - '±'：改变结果符号
  - '='：计算表达式的值
  - 'AC'：清零
  - 'MC'：清空内存值
  - 'M+'：将显示值与内存值相加
  - 'M-'：将内存值减去显示值
  - 'MR'：重新调用存储在内存中的值
  - 'delete'：删除表达式最后一个字符
