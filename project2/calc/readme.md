###功能
1.基本功能已实现,支持优先级运算和之后运算符扩展
2.增加了简单的处理,以防止用户错误操作(eg: 连续输入操作符等)  
3.结果区文字大小自适应缩小,实时显示计算结果   
4.表达式区超长则向左移动   
5.如上图,界面自动布局
